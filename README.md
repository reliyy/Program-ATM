# Program-ATM
Program ATM basic
Di sebuah kota di Indonesia, Hidup seorang pemuda yang tampan dan cerdas yang sangat terkenal hebat ngoding. Karena kehebatannya dalam ngoding, sebuah perusahaan bank yang baru meminta dia untuk dibuatkan program untuk mesin ATM mereka, dengan aturan berikut:
1.	Pengguna harus menginput PIN
2.	Jika PIN benar, akan muncul menu dengan 4 pilihan
3.	Pengguna dapat cek saldo, setor tunai, Tarik tunai, dan cancel 
4.	Jika pengguna memilih cancel, maka program selesai
